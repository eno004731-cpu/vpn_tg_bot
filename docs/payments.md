# Оплата без эквайринга

## Telegram Stars

Telegram Stars уже встроены в бота как отдельный способ оплаты. Для них не нужен `provider_token`: бот отправляет Telegram invoice с валютой `XTR`, получает `successful_payment` и сразу активирует подписку в `3x-ui`.

Настройка лежит в `config/plans.toml`:

- `price_stars` - цена тарифа в звёздах;
- `provision_access = false` - тестовая оплата без выдачи доступа;
- `stars_test` - тестовый пункт за 1 звезду.

## Практичный MVP

Если вы не хотите сразу подключать платёжную систему, то рабочий вариант такой:

1. Бот создаёт инвойс с уникальной суммой.
2. Клиент переводит деньги на карту или по телефону.
3. Бот переводит платёж в статус ожидания.
4. Админ подтверждает оплату и выдачу доступа.

Такой путь не даёт полностью автоматического подтверждения, зато запускается быстро и без договора с платёжным агрегатором.

## Как довести до полуавтомата

Самый простой следующий шаг:

- сохранять уникальную сумму в инвойсе;
- сверять входящий перевод по выписке банка;
- если совпадение одно и сумма уникальна, автоматически подтверждать инвойс.

Для личной карты у банков обычно нет нормального публичного API для входящих переводов физлицу, поэтому без бизнес-счёта или агрегатора приходится жить через выписку, пуши или ручное подтверждение.

## Если нужен сервис вместо ручной схемы

### ЮKassa

Подходит, если нужен более нормальный checkout и дальше возможны автоплатежи. На сайте ЮKassa прямо указаны:

- подключение онлайн;
- приём карт, SberPay, СБП, T-Pay;
- сценарии для подписок и автоплатежей.

Источник: [ЮKassa](https://yookassa.ru/)

### CloudPayments

Подходит, если хочется webhook-события и довольно привычный сценарий с платёжными ссылками.

Источник: [CloudPayments](https://cloudpayments.ru/)

### T-Bank / СБП для бизнеса

Если у вас будет бизнес-счёт, можно смотреть в сторону банковского API и СБП-ссылок. Это уже ближе к автоматической сверке и более чистой бухгалтерии.

Источник: [T-Bank для разработчиков](https://developer.tbank.ru/)

## Что я бы выбрал

- если нужен запуск за 1 вечер: текущая схема `уникальная сумма + подтверждение админом`;
- если нужен следующий шаг без боли: `ЮKassa`;
- если хотите сильнее опираться на банк и СБП: `T-Bank` для бизнеса.
